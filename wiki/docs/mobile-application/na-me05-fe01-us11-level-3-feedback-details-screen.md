| 05/02/2025 draft is created by Galina Lonskaya |### User story

\> As an account owner, I would like to view the Level 3 Feedback screen so that I can set the proper settings for Level 3 Feedback.

Content

User story Acceptance criteria Level 2 Feedback: entry point Level 3 Sound mode adjustment Level 3 Vibration mode adjustment Level 3 Static mode adjustment

### Acceptance criteria

| AC | Text | UI Design | Level 2 Feedback: entry point | Level 3 Sound mode adjustment | Level 3 Vibration mode adjustment | Level 3 Static mode adjustment |
|---|---|---|---|---|---|---|
| AC01 | If I tap on the Level 3 tile at the Feedback Settings screen, then:The Level 3 screen should be displayed with the default or previously chosen mode (Sound or Vibrate or Static) or Default. | Figma |
| AC02 | By default the following Level 3 Feedback settings should be applied:Mode: StaticVolume: 4 | - |
| AC03 | The same as for Level 1, with 2 exceptions:the default sound should be Beep 2x. | Figma |
| AC04 | The following Sound list should be opened for Level 3. | Figma |
| AC05 | The same of Level 1, only exception: By default Double Paw Tap. | Figma |
| AC06 | If I tap on the tile with Vibration, then following Vibration List should be opened. | Figma |
| AC06 | If I tap the Static button, then: the Static Slider Bar should be shown, see (NA) ME05-FE01-US03: Static Bar Slider | Figma |
| NOTE: All other functionality is the same as for Level 1 |
