Background and goalThe need to rename the buttons raised when we decided to put all feedback buttons in one row in IF redesign story. The buttons became smaller and the previous titles didn't fit.

# User story

\> As Halo account owner I want to have shorter titles for encouragements so that I can read fully encouragement title on any screen of the app.
