User Story Acceptance criteria

# User Story

\> As a Halo acount owner I want to be able to resolve autogenerated fence overlaping with old ones by replacing them during fence creation so that I should NOT do it myself by leaving the fence creation and spend less time on fence replacement.

# Acceptance criteria
