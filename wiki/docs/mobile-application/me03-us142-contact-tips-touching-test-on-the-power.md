User Story Acceptance criteria

# User Story

\> As a Halo acount owner I want that I want to be able to check if contact tips touch my dogs skin so that I can be sure that dog feels corrections and returns back in case of danger.

# Acceptance criteria

| AC | Description |
|---|---|
| ME03-US142-AC01 | PreconditionMy pet has a Halo 4 collarThe collar is worn on my petIf I click the Power button on the collar, then the collar should start the contact tips touching test. |
| ME03-US142-AC02 | Contact tips touching test should consist of the following 10 checks:TBD |
| ME03-US142-AC03 | If at least one of the 10 checks failed, then the full contact tips touching test should be considered as failed.Otherwise, the full contact tips touching test should be considered as successful. |
| ME03-US142-AC04 | If the contact tips touching test fails, then both GPS and Battery LED indicators should blink 3 times in red. |
| ME03-US142-AC05 | If the contact tips touching test fails, then both GPS and Battery LED indicators should blink 3 times in green. |
