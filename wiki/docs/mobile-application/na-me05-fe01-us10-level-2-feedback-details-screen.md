| 05/02/2025 draft is created by Galina Lonskaya |### User story

\> As an account owner, I would like to view the Level 2 Feedback screen so that I can set the proper settings for Level 2 Feedback.

Content

User story Acceptance criteria Level 2 Feedback: entry point Level 2 Sound mode adjustment Level 2 Vibration mode adjustment Level 2 Static mode adjustment

### Acceptance criteria

| AC | Text | UI Design | Level 2 Feedback: entry point | Level 2 Sound mode adjustment | Level 2 Vibration mode adjustment | Level 2 Static mode adjustment |
|---|---|---|---|---|---|---|
| AC01 | If I tap on the Level 2 tile at the Feedback Settings screen, then:The Level 2 screen should be displayed with the default or previously chosen mode (Sound or Vibrate or Static) or Default (i). | Figma |
| AC02 | By default the following Level 2 Feedback settings should be applied:Mode: VibrationSound: Double Paw TapVolume: n/a | - |
| AC03 | The same as for Level 1, only exception: By default Double Beep 2. | - |
| AC04 | The same of Level 1, only exception: By default Double Paw Tap. | - |
| AC05 | If I tap on the tile with Vibration, then following Vibration List should be opened. | Figma |
| AC06 | If I tap the Static button, then: the Static Slider Bar should be shown, see (NA) ME05-FE01-US03: Static Bar Slider | - |
| NOTE: All other functionality is the same as for Level 1 |
