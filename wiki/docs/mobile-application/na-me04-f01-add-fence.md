# User story

\> As Halo app account owner, I want to create a fence, so I can keep my pet safe.

# Acceptance criteria

Fence Adding should consist of the following steps depending on autogenerated fence type:

| AC | ME04-F01-AC01 | ME04-F01-AC02 |
|---|---|---|
| Creation steps \Fence type | Fence autogenerated based on cadastral data | Simple square |
| Checks before creation | (NA) ME04-US01. Add Fence: Fence adding limitations |
| Step 0 (the first adding only) | (NA) ME04-US02. Add Fence: Step 0. Fence Tutorial (the very first adding only) |
| Step 1 | (NA) ME04-US03. Add Fence: Step 1. Set Fence LocationIncluding: (NA) ME04-US04. Add Fence: Step 1a. Search location by address |
| Step 2 | (NA) ME04-US05. Add Fence: Step 2. Fence autogeneration attempt |
| Step 3 | (NA) ME04-US07. Add Fence: Step 3b. Confirm Fence | (NA) ME04-US06. Add Fence: Step 3a. Simple square fence edit |
| Step 4 (only if user decided to edit fence before saving) | (NA) ME04-US08. Add Fence: Step 4. Edit Fence (optional) | Absent because user is already in edit mode on Step 3 |
| Step 5 | (NA) ME04-US09. Add Fence: Step 5. Name+Save Fence |Tech details
