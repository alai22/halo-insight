The rules for determining the main building are also described here - Validation of intersections with objects on the map#Buildings

We don't need to use additional data loading, otherwise, it can be done indefinitely. To collect analytics, we use the data that we have already loaded.16 zoom map level (~1 km from fence) in the story - this describes that we get data for 16 zoom maps

Buildings are used without additional buffers to collect analytics. During validation, we use buildings with buffersIn the case of water and roads, both original objects and buffered objects (i.e. polygons) are used

Intersection with other fences is only possible after generation.This only needs to be calculated for (NA) ME04-F??. Fence analytics in Amplitude

Calculating analytics can be part of validation, but you don't need to do the calculation every time you change the fence - you only need to do it before saving the fence
