# User story

\> As Halo app account owner, I want view the post-fence event marker on the map so I know where my pet returned back to safety.

# Acceptance criteria

| AC | Description | Design |
|---|---|---|
| ME04-US21-AC01 | Once fence breach event is finished, the app should:Display the Post-fence event marker on the fence in the location where the pet returned to safety. | Figma |
| ME04-US21-AC02 | The app should display Post-fence event marker after pet returned from both emergency and warning zone. |
| ME04-US21-AC03 | The app should display the Post-fence event marker only if Pet is selected??? |
| ME04-US21-AC04 | The app should hide the Post-fence event markers on the map at the end of events' day (midnight) following timezone of user's device. |
