# User story

\> As a Halo app user I want to have a link to the terms of the Referral Program so that to understand all conditions of it.

# Acceptance criteria

| AC | Description | iOS screen design/ implementation notes | Android screen design/ implementation notes |
|---|---|---|---|
| EC03-US09-AC01 | 'Learn more' link should be added to the "Refer a Friend" screen. | Link to Zeplin | The same as for iOS. |
| EC03-US09-AC02 | When the user taps on 'Learn more' link, then the app should open the 'Referral Program' page in the default mobile browser. | - | - |
| EC03-US09-AC03 | The link to the 'Referral Program' page is: https://www.halocollar.com/referral-program/. | - | - |
| EC03-US09-AC04 | The link to the 'Referral Program Terms' article should be stored on BE side. | - | - |
