# General description

We assume that if there are no fences' centers in the fences overlapping area - then they are probably on different cadastral parcels and we should not push the user to replace their old fence(s) with new ones. It seems that they just try to add a fence on the neighbor's parcel in addition to the fence on their home's parcel.

But in case there are some fences' centers in the fences overlapping area - then the user just probably needs to replace the old fences with a new more accurate one. If not - they can manually adjust the new fence so it fits on the same parcel with already existing fences.

# User Story

\> As a Halo acount owner I want to be able to resolve autogenerated fence overlaping with automatic exclusion of overlapping area if none fences center in it so that I should NOT adjust new fence borders myself and spend less time on handling overlapping.
