|  |# User story

\> As an account owner, I would like to apply an instant feedback so that I can quickly apply prevention or encouragement to train my pet.

Contents

User story Acceptance criteria

### Acceptance criteria

| AC | Text | UI design |
|---|---|---|
|  | Precondition: My Fences tab is openedPet card is shown I have 1 or more pets with the assigned collars in my account. If I tap 'MF' icon, then: MF card with the pet selected should be displayed (the pet to which the pet card). Note for dev: the selection choice for the future reusing can be stored locally on the mobile device. TBD with Ryan |  |
|  | Precondition: My Fences tab is opened I have more then one pet with the assigned collars in my accountdata about the pet selection choice is available. If I tap 'MF' icon, then: MF card with the previously selected pets should be displayed. |  |
|  | Precondition: My Fences is opened and I have more then one pet with the assigned collars in my account no data about last selection choice. If I tap 'MF' button, then: all pets should be displayed as selected. |  |
|  | In case last time when I opened MF card, I deselected all pets, then:when I open MF next time, all pets should be selected by default. |  |
|  | If all pets don't feet into the screen width, then:one icon with number should be shown with the number of pets |  |
