# User story

\> As Halo app account owner, I want to be able to access the Fence Tutorial from Add Fence and Edit Fence screens so I can quickly remind myself how it works if I forgot something.

# Acceptance criteria

| AC | Description | Design |
|---|---|---|
| ME04-F05-AC01 | If I tap the 'Fence Tutorial' button on one of the following screens:(NA) ME04-US03. Add Fence: Step 1. Set Fence Location(NA) ME04-F02. Edit Fence Bordersthen the app should:Open the Fence Tutorial. | Figma |
| ME04-F05-AC02 | If I tap one of the following buttons:either the 'Skip' button on any of tutorial steps,or the 'I'm Ready!' on the last stepthen the app should:Navigate back to one of the following screens :(NA) ME04-US03. Add Fence: Step 1. Set Fence Location(NA) ME04-F02. Edit Fence Borders |
| ME04-F05-AC03 | User can navigate between tutorial screens:either using swipe left/right,or the 'Next' button. |
