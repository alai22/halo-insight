| HALO-2140 - The MapView should centered on CollarPin during Create Fence with collar flow Closed |# User story

\> As owner, I want to view a map centered on the collar pin in order not to meet the situation when the collar move away and I am forced to look for it ( relevant for creating/ editing a fence).

# Note

It was discussed with Michael within the BA call that the following behavior is preferable: We need to add the "Center on user's location" button to the "Create New Fence/Edit fence posts" screen. If the button is pressed, then the map should be centered on the user location after each location update, if not the map isn't centered automatically.
