User story Acceptance criteria

# User story

\> As an Owner I want to connect to Wi-Fi networks with different security protocols so that I could use my collars with different types of Wi-Fi networks.

# Acceptance criteria

| AC | Text | Links / Notes / Designs / Wireframes |
|---|---|---|
| ME03-US10-AC01 | The mobile app and collar support working with open Wi-FI networks. | N/A |
| ME03-US10-AC02 | The collar communicates a security protocol type for every Wi-Fi network to the mobile application. The WEP, WPA and WPA2 networks are displayed with the corresponding icon, while open Wi-Fi networks are displayed without it. |  |
| ME03-US10-AC03 | Users can see RSSI next to every Wi-Fi network in the list. | N/A |
| ME03-US10-AC04 | When the user selects a WEP/ WPA/ WPA2 network the mobile app offers him/her to select the Wi-Fi network and enter the password for it. | N/A |
| ME03-US10-AC05 | When the user selects an open network the mobile app attempts to connect to it without the password. | N/A |
| ME03-US10-AC06 | TBD: the mobile app notifies the user that the collar does not support working with open networks with web page authentication | TBD: screen design |
