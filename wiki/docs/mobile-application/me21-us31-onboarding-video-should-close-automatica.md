User story Acceptance criteria

# User story

\> As a Halo app user I want a video to close automatically after ending so that I don't need to do extra taps and close it myself.

# Acceptance criteria

| AC | Description | iOS screen design | Android screen design |
|---|---|---|---|
| ME21-US31-AC01 | Onboarding video in a full-screen mode should close automatically after it ends.BA note: if a user just scrolled the video till the end, the app will also consider this case as watching till the end and close automatically. | - | - |
| ME21-US31-AC02 | The same requirements should be also applied for prompting videos on the following screens:'You need a Plan' screen;'Your Trial Has Expired' screen. | - | - |
| ME21-US31-AC03 | The user should still be able to close the abovementioned videos by themselves before ending. | - | - |
