User story Acceptance criteria Change timeout for applying feedback for a single pet Change timeout for applying feedback for multiple pets

# User story

\> As an owner, I want to find out that a feedback command didn't get trhough within couple of seconds because if it takes too long to appy a feedback, the dog may not understand what is wrong and why the command was sent.
