# User story

\> As a Halo account owner I want to switch between different generations of the collar on the instruction screens so that I can see the relevant instructions for my specific collar model.

# Acceptance criteria

| AC | Description | UI design |

| ME21-US74-AC03 | If I tap on Older button, then: 'Older' button should become selectedCorresponding image/GIF should be displayed |
| ME21-US74-AC04 | If I tap on Halo Collar 4 button, then: 'Halo Collar 4' button should become selectedCorresponding image/GIF should be displayed |
