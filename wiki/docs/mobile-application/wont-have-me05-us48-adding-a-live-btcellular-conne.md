| HALO-10502 - MOB: Add a live BT/cellular connection indication to the feedback card Closed |# General description

Based on real-life experience, we should provide either a prompt and/or on-screen warning before you can send an Instant Feedback to a dog that is \> some distance from the phone (based on the GPS location of each). This is the make sure you realize that this dog is not nearby, and you:a. May be sending the feedback to the wrong dog (if you have \> 1 pet); OR,b. Should be aware that dogs may not understand remote instant feedback

Upd 06/09/2022 the functionality will be implemented in scope of [Won't have] ME05-EP03. 2022 Instant feedback UI/UX redesign

# User story

\> As an app owner, I want to see the on Apply Feedback for \<Pet Name\>' screen an indication of which connection is used between me and my dog so I will not send the instant feedback commands if the dog is not in a Bluetooth range
