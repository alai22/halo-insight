# User Story

\> As a user, I want to receive different notifications for each specific case when I can't send a command to my Halo collar so that I can understand what is exactrly the problem. At the moment the mobile app shows 'Collar is out of Wi-Fi, cellular and Bluetooth range' if it fails to send a command to a collar due to any reason. As the result it seems that the device does not work properly and Halo Team receves a lot of complaints.
