\> As Halo app account owner I want to rename vibrations available for feedback setting so that it can be made more intuitive for users to select during feedback setting.

# Acceptance criteria

| AC | Description |  |
|---|---|---|
| ME05-US37-US01 | The following vibrations should be renamed: ME05-US37-ID01 – 06, see 'New title' and 'Old title' columns in Table 1. Updated vibration list below. | Figma |
| ME05-US37-US02 | The vibration sequence should be the same as presented in ME05-US37-ID01 – ID06. Note about backward compatibility: agreed with the dev team that it's OK that the new sequence will be applied both for old and new mob versions. |
