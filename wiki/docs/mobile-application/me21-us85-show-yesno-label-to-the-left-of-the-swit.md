| 09 Jul 2024 draft user story is created |# User story

\> As Halo mob app user, I want to see a clear 'yes/no' label to the left of the switcher, so that I can easily understand the options available and make an informed choice without confusion.

# Acceptance criteria

| AC | Description | UI design |
|---|---|---|
| ME21-US85-AC01 | Precondition: 'Tell us about yourself' screen is opened. If 'Have you trained dogs before?' switcher is turned OFF: then 'No' label in grey should be displayed to the left of the switcher. | Figma |
| ME21-US85-AC02 | Precondition: 'Tell us about yourself' screen is opened. If 'Have you trained dogs before?' switcher is turned ON: then 'Yes' label in blue should be displayed to the left of the switcher. | Figma |
