| 07 Mar 2024 Draft version is created by Galina Lonskaya |# User story

\> As a Halo app account owner, I want to have a button within the app that directly takes me to my phone's settings, so that I can easily opt in or opt out of receiving push notifications.

# Acceptance criteria

| AC | Text | Links / Notes / Designs / Wireframes |
|---|---|---|
| ME02-US37-AC01 | Push Notifications button should be added to Settings screen next to My Account button. | Zeplin/Figma link |
| ME02-US37-AC02 | Precondition: I have version iOS 15.4 and higher OR Android 9.0 or higher. If I tap on Push Notifications button on Settings screen, then: Halo app should redirect me to Halo notification settings in my phone's settings where I can manage push notifications of Halo app. | - |
| MEXX-US37-AC03 | Precondition: I have version iOS 15.3 and lower OR Android 9.0 or lower.If I tap on Push Notifications button on Settings screen, then: Halo app should redirect me to the list of Halo app permissions in my phone's settings. | - |# Implementation notes
